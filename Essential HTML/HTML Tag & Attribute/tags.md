###  **Common HTML Tags**

| **Tag** | **Description** |
|----------|-----------------|
| `<html>` | Root element of a webpage. Everything (except `<!DOCTYPE html>`) goes inside this tag. |
| `<head>` | Contains document information such as metadata, title, etc. |
| `<title>` | Sets the title of the webpage (appears on browser tab). |
| `<body>` | Contains the visible content of the document (text, images, links, etc.). |
| `<h1>` â€“ `<h6>` | Define headings, from largest (`<h1>`) to smallest (`<h6>`). |
| `<p>` | Defines a paragraph of text. |
| `<a>` | Creates a hyperlink to another page or resource. |
| `<img>` | Embeds an image in the document. |
| `<ul>` / `<ol>` | Create unordered (bulleted) and ordered (numbered) lists. |
| `<table>` | Defines a table. |
| `<div>` / `<span>` | Generic containers for grouping or styling elements. |
| `<form>` | Creates a user input form. |
| `<input>` | Defines an input field within a form. |
